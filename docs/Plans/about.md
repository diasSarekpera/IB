# 🔥 🖼️ BLOC GAUCHE (VISUEL PREMIUM)

## 🎯 Objectif :

Créer **impact + crédibilité + élégance**

---

## 💎 Composition

### 1. Image principale

* format : portrait (ratio 4:5)
* largeur : ~420px
* border-radius : **16px**
* box-shadow :

  ```
  0 20px 60px rgba(0,0,0,0.08)
  ```

---

### 2. Cadre graphique (ton idée améliorée)

👉 2 formes en “L” autour de l’image :

* épaisseur : **6px**
* couleur : **#d4af37 (or)**
* opacité : **0.6**
* position :

  * coin haut gauche
  * coin bas droit

👉 effet :

* légèrement décalé (offset 10–15px)
* donne un effet “design éditorial premium”

---

### 3. Background subtil

👉 derrière le bloc :

* fond blanc cassé ou très léger dégradé :

  ```
  linear-gradient(135deg, #ffffff 0%, #f9f9f9 100%)
  ```

* motif floral :

  * opacity : **0.03 – 0.05**
  * blur léger
  * position : top-left

👉 invisible mais présent = niveau premium

---

### 4. Micro effet

👉 hover image :

* scale : 1.02
* transition : 0.4s ease
